# Markdown to HTML Converter

Converts a Markdown (`.md`) file into a styled, standalone HTML file. Supports headers, bold/italic text, inline code, fenced code blocks, links, unordered and ordered lists, blockquotes, and paragraphs. A handy companion to your Blog Generator project.

## Examples

```bash
python main.py notes.md
python main.py notes.md -o notes.html --title "My Notes"
```
